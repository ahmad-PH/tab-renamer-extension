Welcome to Tab Renamer!

Customize your tab titles and favicons for a more organized browsing experience.

How to Use Tab Renamer

  1. The main way to use the extension is to trigger the UI via a keyboard shortcut. The default is Ctrl+Shift+Y or Command+Shift+Y on Mac, but you should double check, sometimes this is not set on install as it might conflict with existing shortcuts. Make sure to set a shortcut you are comfortable with.
  2. You can also trigger the extension UI by right clicking anywhere on your browser's page, and selecting 'Rename Tab' item from the context menu.
